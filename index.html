HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element appears as the tab name.
  • The head element is a container for metadata.
  • The meta element helps with the search engine and is used by browsers.

CSS

  • Allows for creativity and styling of the website.
  • Allows for font styling.
  • Allows for wallpaper colouring or using images as wallpapers.
  • Can set up a class or id to set a whole bunch of stuff to a certain style.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • To link a stylesheet you put in the head element: link rel="stylesheet" href"assets location"

Git

  • Helps clone and push repos and stores code online.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

Javascript

  • Allows for code that looks for and digests the user's actions, such as hovering their cursor over something or clicking a button.
  • Allows for interactivity with the website.
  • Can be used to set up a trivia website.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
